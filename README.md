## âš¡ Getting Started

### Prerequisites
* **Node.js** (v18 or higher recommended)
* **npm** or **Yarn**

### Quick Setup

1. **Clone & Enter**
   ```bash
   git clone https://github.com/omartantawy360/edu-por-3.git
   cd edu-por-3
   ```

2. **Install & Launch**
   ```bash
   npm install
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
